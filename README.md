## Installation

1. Créez un environnement virtuel :

   ```sh
   python -m venv .env
   ```

2. Activez l'environnement virtuel :

   ```sh
   source .env/bin/activate
   ```

3. Installez les dépendances :

   ```sh
   pip install -r requirements.txt
   ```
